What this site is about

HyperText Transfer Protocol (HTTP) is the baseline application-layer protocol of the Web. It uses a simple request–response model so different devices can communicate in a predictable way. This site summarizes my essay on HTTP: how requests and responses work, how caching saves time, where HTTP came from, and where it’s going.
