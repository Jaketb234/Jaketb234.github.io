About Me & The Project

This project was created by Jake Wright for IT 3203 - Intro to Web Development in the Spring 2025 semester. The purpose of this project is to demonstrate knowledge of HTML and CSS by creating a multi-page website about the HyperText Transfer Protocol (HTTP). Each page covers a different aspect of HTTP, including its semantics, caching mechanisms, history, and future. The website is designed to be simple and easy to navigate, with a consistent layout and style across all pages. The project showcases the ability to create structured content using HTML elements such as headings, paragraphs, lists, and links, as well as applying CSS for styling and layout. Overall, this project serves as a practical application of web development skills learned in the course.
